# 🌪️ TIPPING POINT
### *The Future is Negotiated.*

**Tipping Point** is a high-stakes geopolitical simulation where structural power, economic theory, and human friction collide. You are not just a player; you are a function of the state.

## ⚙️ Nano Season 1: The Diplomacy Summit
* **Cultivate Diplomatic Power:** Establish global influence through strategic alliances.
* **Broker Trade Agreements:** Manage the delicate balance of IPE and macro-stability.
* **Mediate Regional Conflict:** Navigate the "Tipping Point" before the collapse.

## 🛠️ Built With
* **Logic:** JavaScript (Crisis Loop Mechanics)
* **Design:** Functional Aestheticism (CSS3)
* **Context:** International Political Economy Patterns
